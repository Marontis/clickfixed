Simulation Sandbox

Click Fixed Interceptor

Test the extension's ability to intercept DOM clipboard-write operations and classify them using Gemini.

How to Load & Test:
  • Open Chrome and navigate to chrome://extensions
  • Enable "Developer mode" in the top-right corner.
  • Click "Load unpacked" and select the extension folder in this project root.
  • Pin the extension, open this page in Chrome, and click the buttons below.
Scenario 1: Legitimate Copy

Copies normal, safe text to the clipboard. The regex heuristic and Gemini model should evaluate this as safe, allowing the write to complete immediately.

Scenario 2: Standard Attack

Simulates a malicious script injecting a PowerShell cradle designed to bypass execution policy and download remote malware. The extension should **block** the write and show a warning panel.

Scenario 3: Obfuscation Test

Simulates an obfuscated attack payload using carets (c^m^d) and backticks (p`o`w`e`r). Normalization should strip these tricks and flag the threat.

Scenario 4: ClipboardItem API

Attempts to write malicious text using the newer navigator.clipboard.write() API with ClipboardItem blobs, attempting to bypass standard writeText hooks.

Scenario 5: Proactive Modal Lure

Simulates a dynamic modal injection containing a ClickFix captcha verification lure. The extension's MutationObserver should detect and block it proactively.
